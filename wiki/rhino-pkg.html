    Rhino Linux | rhino-pkg Wiki

    rhino-pkg is our custom package management wrapper written in Bash. It was designed with simplicity and ease of use in mind. It will allow for you to search for, and then install packages from any packagem management repository (apt, Pacstall, Snap and Flatpak).

    • It is strongly recommended to utilise rhino-pkg, rather than each package manager individually.

    Our utility, rhino-pkg, will allow for you to also update all of your applications, across multiple different package managers, with easy-to-read terminal output to help improve your experience whilst using the distribution.

    Commands

    To get started with rhino-pkg you will want to ensure that all of your packages are currently up to date.

    rhino-pkg update - This command will search through Apt, Pacstall, Flatpak and Snap and see if any packages require updating.

    rhino-pkg search [package] - This command will search all package managers for what you are looking for and display an output.

    rhino-pkg install [package] - This command will search all package managers for the selected package name, display an output. You can then select the corrosponding package.

    rhino-pkg remove [package] - This command will search all package managers for the selected package which is installed, you can then select the corrosponding package that you wish to remove.

    • You can also call rhino-pkg by just typing rpk instead.
